## Purpose 
* This is a very simple scaffolded ruby on rails app, that is a companion to medium write up.

## Dependencies 
* Ruby version: `2.3.1`
* postgresql
* Rails 
* bundler

## Running 
* `git clone`
* `bundle install`
* `rake db:create`
* `rake db:migrate`
* `rake db:seed`
